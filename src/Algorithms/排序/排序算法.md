## 什么是排序算法？
将无序变为有序

O(n) + O(n) = O(n)
O(lgn) + O(n) = O(n)

+ 复杂度是度量指标随着输入规模增章关系的一种分类。描述的是随着输入规模增加算法中最大的影响因子。
+ 数学上O(n)是指随着规模增长，算法的执行时间会在T=cn内波动，C是大于0的任意常数
+ BIG-O 是一种渐进记号

## 按复杂度分类
+ O(n^2): 插入排序、比较排序、冒泡排序
+ O(nlgn): 合并排序、快速排序、分块排序
+ O(n): 桶排序、基数排序

## 按原地/非原地
原地：插入排序、比较排序、冒泡排序、快速排序
非原地：合并排序、桶排序、分块排序、基数排序
